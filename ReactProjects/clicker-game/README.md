## 🏁 Getting Started

### 📦 Prerequisites

- Node.js (LTS version recommended): [https://nodejs.org/](https://nodejs.org/)
- npm (comes with Node)

### ⚙️ Installation & Run

```bash
# Clone the repository
git clone https://github.com/vishnuskandha/clicker-game-react.git

# Navigate to project directory
cd clicker-game-react/ReactProjects/clicker-game

# Install dependencies
npm install

# Start the app
npm start
